## Summary
In this project, multi section landing pages are created which dynamically add content to a web page as one would not know the amount of content in a particular page which creates a dynamic experience for the user. This project focuses on the concept of landing page. The page is divided into different sections. When the user clicks on a section, the frame of the page navigates to that particular section. It also highlights the current section of the page. This project consists of the HTML and CSS for styling. Javascript is used to provide dynamic experience.
